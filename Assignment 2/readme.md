# Python Programming Practicals

## Overview

This repository contains basic Python programs demonstrating conditional statements, dictionaries, and file handling operations.

## Programs

### 1. Grade Checker

Checks and displays grades based on marks entered by the user using `if-elif-else`.

### 2. Student Grades Management

Uses a dictionary to add, update, and display student grades.

### 3. Write to a File

Creates a text file and writes content into it using file handling functions.

### 4. Read from a File

Reads and displays the contents of a text file.

## Technologies Used

* Python 3.x

## Learning Outcomes

* Conditional Statements
* Dictionary Operations
* File Handling
* Basic Python Programming
